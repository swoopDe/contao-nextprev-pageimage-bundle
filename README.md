This bundle extends Contao with a Next/Previous navigation as Frontend Modul that can display page images alongside the navigation links.

It is designed to work seamlessly with both Contao 4.13 and Contao 5.x, supporting Twig and HTML5 templates.
###  Description
* Next/Previous navigation for pages
* Display of page title and page image
* Fallback handling (e.g. when no page image is set)
* Custom CSS classes and flexible wrapper structure for easy styling
* Compatible with Contao 4 and 5 (Twig & HTML5)
